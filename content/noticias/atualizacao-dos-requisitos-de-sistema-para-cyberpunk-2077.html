Atualização dos requisitos de sistema para Cyberpunk 2077

A CD Projekt Red atualizou os requisitos de sistema para o Cyberpunk 2077, tanto para a atualização 2.0 quanto para a próxima expansão Phantom Liberty. A nova atualização só suportará SSD, então libere espaço ou compre um se precisar trocar.

Os requisitos de sistema atualizados são significativamente mais pesados do que os requisitos mínimos anteriores. Agora você precisará de um mínimo de 12GB de RAM e 6GB de VRAM para rodar o jogo em configurações baixas, com uma taxa de quadros esperada de 30. Os requisitos mínimos atuais estão próximos dos requisitos recomendados do jogo no lançamento. Rodar o jogo em resoluções mais altas do que 1080p ou com ray-tracing exigirá suas próprias especificações de hardware. Os requisitos completos do sistema estão listados abaixo.

Especificações revisadas para Cyberpunk 2077

Requisitos sem ray-tracing

Mínimo

  • Configuração gráfica no jogo: baixa
  • Resolução: 1080p
  • Taxa de quadros esperada: 30
  • Sistema operacional: Windows 10 de 64 bits
  • Processador: Core i7-6700 ou Ryzen 5 1600
  • Placa de vídeo: Geforce GTX 1060 6GB ou Radeon RX 580 8GB ou Arc A380
  • VRAM: 6GB
  • RAM: 12GB
  • Armazenamento: 70GB SSD

Recomendado

  • Configuração gráfica no jogo: alta
  • Resolução: 1080p
  • Taxa de quadros esperada: 60
  • Sistema operacional: Windows 10 de 64 bits
  • Processador: Core i7-12700 ou Ryzen 7 7800X
  • Placa de vídeo: Geforce RTX 2060 Super ou Radeon RX 5700 XT ou Arc A770
  • VRAM: 8GB
  • RAM: 16GB
  • Armazenamento: 70GB SSD

Ultra

  • Configuração gráfica no jogo: ultra
  • Resolução: 2160p
  • Taxa de quadros esperada: 60
  • Sistema operacional: Windows 10 de 64 bits
  • Processador: Core i9-12900 ou Ryzen 9 7900X
  • Placa de vídeo: Geforce RTX 3080 ou Radeon RX 7900 XTX
  • VRAM: 12GB
  • RAM: 20GB
  • Armazenamento: 70GB NVME

Requisitos com ray-tracing

Mínimo

  • Configuração gráfica no jogo: ray tracing baixo
  • Resolução: 1080p
  • Taxa de quadros esperada: 30
  • Sistema operacional: Windows 10 de 64 bits
  • Processador: Core i7-9700 ou Ryzen 5 5600
  • Placa de vídeo: Geforce RTX 2060 ou Radeon RX 6800 XT ou Arc A750
  • VRAM: 8GB
  • RAM: 16GB
  • Armazenamento: 70GB SSD

Recomendado

  • Configuração gráfica no jogo: ray tracing ultra
  • Resolução: 1080p
  • Taxa de quadros esperada: 60
  • Sistema operacional: Windows 10 de 64 bits
  • Processador: Core i9-12900 ou Ryzen 9 7900X
  • Placa de vídeo: Geforce RTX 3080Ti ou Radeon RX 7900 XTX
  • VRAM: 12GB
  • RAM: 20GB
  • Armazenamento: 70GB NVME

Overdrive com ray-tracing

  • Configuração gráfica no jogo: ray tracing overdrive
  • Resolução: 2160p
  • Taxa de quadros esperada: 60
  • Sistema operacional: Windows 10 de 64 bits
  • Processador: Core i9-12900 ou Ryzen 9 7900X
  • Placa de vídeo: Geforce RTX 4080
  • VRAM: 16GB
  • RAM: 24GB
  • Armazenamento: 70GB NVME

A CD Projekt Red também recomenda desinstalar quaisquer mods antes de instalar a versão 2.0. As alterações no jogo base são abrangentes o suficiente para que qualquer mod possa causar problemas de compatibilidade e glitches. É melhor esperar até que os mods sejam atualizados para se adequarem à versão 2.0. O desenvolvedor também recomenda verificar seus sistemas de resfriamento antes de jogar Phantom Liberty, pois a expansão pode ser intensiva para a CPU.

Na análise do GameSpot sobre Cyberpunk 2077: Phantom Liberty, o revisor Michael Higham deu à expansão uma nota 10/10. A atualização 2.0 será lançada em 21 de setembro. Phantom Liberty será lançado em 26 de setembro.
