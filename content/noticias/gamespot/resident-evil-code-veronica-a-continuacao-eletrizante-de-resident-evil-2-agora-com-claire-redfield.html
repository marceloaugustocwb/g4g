Conforme foi relatado pela primeira vez no GameSpot News, Resident Evil: Code Veronica (chamado de Biohazard: Code Veronica no Japão) se passa três meses após o fim de Resident Evil 2, com o jogador mais uma vez controlando Claire Redfield. Após voar para a Europa para investigar o desaparecimento de seu irmão Chris, Claire é sequestrada e levada para uma ilha tropical isolada onde alguns amigos podres e familiares começam a aparecer novamente.

Veronica apresenta uma ruptura significativa com os títulos anteriores da série - todos os cenários agora são feitos em polígonos completos, permitindo que a câmera se mova, acompanhando a ação e uma visão em primeira pessoa. Também há possíveis mudanças ambientais, como tetos desabando ou pilares caindo. Veronica também faz uso dos efeitos especiais do hardware do Dreamcast, como iluminação dramática e sombras e personagens com pele para que você não veja as bordas poligonais serrilhadas.
