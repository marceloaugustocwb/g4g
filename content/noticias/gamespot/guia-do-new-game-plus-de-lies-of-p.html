Tudo o que você precisa saber sobre o modo New Game Plus de Lies of P

O modo New Game Plus de Lies of P fica disponível após assistir aos finais e cenas pós-créditos. Assim que as cenas terminarem, uma mensagem aparecerá perguntando se você deseja sobrescrever seu save com uma nova jogada no modo New Game Plus.

Seu save original: sobrescrever, criar um backup ou reiniciar

Sobrescrever - Se você decidir aceitar a mensagem mencionada acima, você voltará para a área inicial da campanha no modo New Game Plus. Um arquivo de save separado não será criado. Em vez disso, seu save original será sobrescrito.

Criar um backup - Aqueles que jogam no PC via Steam podem optar por criar um backup do save. Seu arquivo de save pode ser encontrado na pasta de instalação do Lies of P: Steam\Steamapps\common\Lies of P\LiesofP\Saved\SaveGames\<userID>. Copie os arquivos de conta, opções e dados de save para uma pasta de backup.

Uma boa razão para criar um backup é para que você possa encontrar outras vantagens que possa ter perdido durante a campanha. Por exemplo, você pode comprar certas armas ou desbloquear conquistas específicas, em vez de fazer essas tarefas no New Game Plus.
